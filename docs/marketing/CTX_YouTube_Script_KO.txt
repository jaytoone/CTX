  1. "Claude Code 기억 잃는 문제 — 훅 하나로 해결했습니다 (회상률 0.88, 10만 턴 실측)"  ← 주력
  2. "Claude Code에 영구 메모리를 붙였습니다 — 세션 닫아도 컨텍스트 유지"
  3. "Claude Code 훅 튜토리얼 — 한 줄 설치로 크로스 세션 메모리"

썸네일 컨셉
  · 발표자 얼굴 우상단
  · 대형 텍스트: "0.00 → 0.88 메모리 회상" (어두운 배경, 밝은 글씨)
  · A/B 대안: before/after 터미널 스플릿 + 얼굴

설명란 첫 줄
  Claude Code는 세션을 닫으면 모든 걸 잊습니다.
  10만 턴 실측 데이터로 검증한 훅으로 그 문제를 해결했습니다.
